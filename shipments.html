Heavy Haul Shipment Project

Heavy Haul Shipment dashboards created via Tableau during Intense Data Analytics Course provided by IntuCraft.

Goal

The main aim of the dashboard is to view the distribution of the heavy duty shipments in USA and the corresponding profit per city.

Results

The dashboard indicates that the highest profit was scored at NewYork. Further, Office supplier has been marked as highest requested order.

Project Duration

This project was accomplished within 3-4 hours during the intense course of Data Analysis provided by Intucraft.
