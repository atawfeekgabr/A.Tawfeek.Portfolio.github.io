Tools

I employ a diverse set of tools to streamline data analysis and visualization:
- Tableau: For creating intuitive, interactive dashboards that simplify complex data insights for stakeholders.
- Python: Leveraging its libraries (Pandas, Matplotlib, Seaborn, Scikit-learn, bootstrap, Numpy) for data processing, visualization, and machine learning to extract and analyze insights.
- SQL: Essential for efficient data retrieval, manipulation, and preparation from relational databases through complex queries.
- PowerBI: Utilized for its seamless integration with Microsoft products, enabling advanced data modeling, analysis, and comprehensive dashboard creation.
- Excel: For preliminary data analysis, manipulation, and visualization, using functions, pivot tables, and charts for quick insights and sharing with less technical users.
Together, these tools allow me to cover the full spectrum of data analysis, from cleaning and exploration to predictive modeling and presentation.
